Creating Multiple Folders From a Bash Array

Sep 11, 2021

This snippet allows you to create a series of directories, that are a part of a regular bash array. It is useful if you have a number of directories to create and without using a loop.

#!/bin/bash


var="eins"
var2="zwei"

play_array=("$var" "$var2")

mkdir "${play_array[@]}"

Directories created:

.
├── eins/
└── zwei/
